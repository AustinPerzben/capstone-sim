CP: Interactive Simulation

About the Project

This simulation is a tool to explore the dynamics of information diffusion in social networks.

The simulation is based on a network of agents, each of which has a certain number of memes. The memes are represented by circles, and the agents are represented by squares. The agents can communicate with each other through channels, which are represented by lines. The agents can also communicate with each other through memes. When an agent is exposed to a meme, it will adopt it with a certain probability. The probability of adoption depends on the number of memes the agent already has, and the number of memes the agent is exposed to.

Controls

Keys - SPACE to toggle Edit and Inspect Mode

Mouse - Click to add Regions (Edit Mode)
Mouse - Drag Regions to move them (Inspect Mode)
Mouse - Hover over objects to learn about them

Scenarios

Explore different scenarios to learn from:

  • Cultural Delays - Test how far a Meme can travel, if given enough time
  • Media Isolation - Investigate how a lack of access to popular Channels affects small Regions

Limitations

  1. This simulation is not a model of reality. It is a tool to explore the dynamics of cultural information transfer through media in macro-social system. It is not meant to be used as a predictive tool.
  2. You are likely to find a few bugs while exploring this site. I'm doing my best to find and fix them all, but some still escape me! Please submit a bug report if you think something is not working the way it should!
