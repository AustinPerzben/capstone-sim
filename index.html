About the Project

The background to this entire site is the canvas for a playable simulation to better understand the relationship between global media and the spread of cultural information across regions and over time.

The simulation is meant to help you understand the relationship between global media and cultural information flow. Memetics is adopted as the representational framework for the macro-social model implemented in the simulation. The project seeks to identify factors that influence the spread of cultural information, investigate the mechanisms that lead to emergent phenomena. You're encouraged to experiment anc explore different scenarios.

Memes are color-coded to represent the Channel they travel through.

Controls

Keys - SPACE to toggle Edit and Inspect Mode
Keys - ARROW KEYS to move the camera

Mouse - Click to add Regions (Edit Mode)
Mouse - Drag Regions to move them (Inspect Mode)
Mouse - Hover over objects to learn about them

Limitations

  1. This simulation is not a model of reality. It is a tool to explore the dynamics of cultural information transfer through media in macro-social system. It is not meant to be used as a predictive tool.
  2. You are likely to find a few bugs while exploring this site. I'm doing my best to find and fix them all, but some still escape me! Please submit a bug report if you think something is not working the way it should!
