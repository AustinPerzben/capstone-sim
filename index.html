CP: Interactive Simulation

About this Site

This simulation is a tool to explore the dynamics of information diffusion in social networks.

The simulation is based on a network of agents, each of which has a certain number of memes. The memes are represented by circles, and the agents are represented by squares. The agents can communicate with each other through channels, which are represented by lines. The agents can also communicate with each other through memes. When an agent is exposed to a meme, it will adopt it with a certain probability. The probability of adoption depends on the number of memes the agent already has, and the number of memes the agent is exposed to.
